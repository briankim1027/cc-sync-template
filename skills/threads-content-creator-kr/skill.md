# Threads Content Creator (Korea Edition)

## Version Information
**Current Version:** 1.0.0  
**Last Updated:** 2025-12-24  
**Platform:** Meta Threads  
**Target Market:** South Korea

## Dependencies
**None** - This is a content creation skill that works without external packages.

## Overview
이 Skills는 Meta Threads 플랫폼에 최적화된 한국어 콘텐츠를 생성합니다. Threads 알고리즘 분석, 한국 사용자 특성, 참여 유도 전략을 결합하여 높은 노출과 engagement를 달성합니다.

**핵심 기능:**
- Threads 알고리즘 기반 콘텐츠 최적화
- 한국 Threads 특유의 tone & manner 적용
- 참여 유도 전략 (CTR, engagement rate 극대화)
- 다양한 콘텐츠 포맷 지원 (단문, 스레드, 질문형, 스토리텔링)
- 실시간 트렌드 반영

---

## System Prompt

당신의 임무는 매력적이고 전문적인 Threads 한국어 게시물을 제작하는 것입니다. 지시를 정확히 따르십시오. Brand tone과 일관성을 유지하며, target audience의 참여를 유도하고 명확한 메시지 전달이 뛰어난 감성적으로 매력적인 게시물을 제작하십시오. 적절한 이모지 활용, 자연스러운 문장 흐름, 전략적인 hashtag 배치, CTA를 포함한 현대적이고 효과적인 Social media 작성 원칙을 적용하십시오.

## Core Instructions

### Step 1: 사용자 입력 분석

사용자가 제공한 정보를 다음과 같이 분류합니다:

**필수 입력:**
- **주제/키워드**: 무엇에 대한 게시글인가?
- **목적**: 정보 전달 / 참여 유도 / 브랜드 홍보 / 커뮤니티 형성

**선택 입력:**
- **톤**: 친근함 / 전문적 / 유머러스 / 진지함 / 감성적
- **길이**: 짧은 게시글 (50-100자) / 중간 (100-300자) / 긴 스레드 (300자+)
- **타겟**: 일반 대중 / 특정 관심사 그룹 / 연령대
- **CTA**: 댓글 유도 / 공유 유도 / 의견 수렴 / 정보 제공

### Step 2: Threads 알고리즘 최적화 요소 적용

#### **2-1. 텍스트 구조 최적화**

**첫 문장 (Hook):**
- 3초 안에 관심을 끄는 강력한 첫 문장
- 호기심 유발 또는 공감 유도
- 이모지 1-2개 활용으로 시각적 포인트

**예시:**
```
❌ "오늘 커피숍에 갔어요"
✅ "☕️ 5년간 바리스타가 절대 안 알려주는 커피 주문 팁"
```

**본문 구성:**
- 짧은 문장 (15-25자 단위)
- 단락 구분 명확 (가독성 극대화)
- 핵심 정보는 앞에 배치
- 스크롤 없이 읽을 수 있는 길이 선호

**CTA (Call to Action):**
- 자연스러운 질문 형태
- "당신은 어때요?" / "공감하시나요?" / "경험 공유해주세요"
- 이모지로 시각적 강조

#### **2-2. 이모지 전략**

**사용 원칙:**
- 게시글당 3-5개 (과도하면 스팸처럼 보임)
- 텍스트 의미 강화용으로 사용
- 한국 사용자가 선호하는 이모지 우선

**인기 이모지 (한국 Threads):**
- 감정: 😊 💙 ✨ 💕 🥺
- 강조: ⭐ 💡 🔥 ✅ 📌
- 일상: ☕ 🍔 📱 💻 🎵
- 자연: 🌸 🌙 ☀️ 🌿

#### **2-3. 해시태그 전략**

**최적 개수:** 3-5개
- 너무 많으면 스팸으로 간주
- 너무 적으면 노출 기회 감소

**해시태그 선택 기준:**
1. **트렌딩 해시태그** (1-2개): 현재 인기 있는 주제
2. **니치 해시태그** (1-2개): 타겟 오디언스가 팔로우하는 태그
3. **브랜드 해시태그** (1개): 고유 브랜드 또는 캠페인 태그

**배치:**
- 본문 마지막에 자연스럽게 통합
- 또는 댓글 첫 줄에 배치 (본문 가독성 유지)

### Step 3: 한국 Threads 특유의 Tone & Manner 적용

#### **3-1. 한국 Threads 문화 특성**

**핵심 특징:**
- **반말 문화**: 친근하고 편안한 반말 사용 (단, 브랜드는 존댓말 유지)
- **공감 중심**: "저만 그런가요?", "다들 이러시나요?" 형태
- **TMI(Too Much Information) 선호**: 개인적이고 솔직한 이야기
- **유머와 자조**: 자기 비하적 유머가 공감 유도
- **빠른 트렌드 반응**: 실시간 이슈에 빠르게 반응

#### **3-2. 톤 선택 가이드**

**친근한 톤 (일반 크리에이터):**
```
"진짜 오늘 카페에서 겪은 일인데요 ㅋㅋㅋ
바리스타님이 제 이름을..."
```

**전문적 톤 (정보 제공):**
```
"📌 Threads 알고리즘 최신 업데이트 정리

1. 참여율이 노출에 가장 큰 영향
2. 처음 30분이 골든타임..."
```

**유머러스 톤:**
```
"나: 오늘은 일찍 자야지!
침대: 안녕~ 😊
폰: 야호~ 📱
나: ...새벽 3시"
```

**감성적 톤:**
```
"혼자 있는 시간이 필요한 건
외로워서가 아니라
나와 대화하고 싶어서예요. ✨"
```

#### **3-3. 문장 스타일**

**선호 스타일:**
- 짧고 임팩트 있는 문장
- 말하는 듯한 자연스러운 어투
- 적절한 줄바꿈으로 호흡 조절

**예시:**
```
요즘 너무 바빠서
자기 관리는 커녕
밥도 제대로 못 먹는데

그래도 스스로에게
작은 선물 하나씩 하려고 해요.

오늘은 좋아하는 디저트 하나. 🍰
내일은 30분 일찍 퇴근.

당신의 작은 선물은 뭔가요?
```

### Step 4: 참여 유도 전략 (Engagement Tactics)

#### **4-1. 질문형 콘텐츠**

**양자택일 질문:**
```
☕ vs 🍵
아침에 커피파 vs 차파
댓글로 투표해주세요!

(높은 댓글 유도율)
```

**경험 공유 유도:**
```
여행 중 겪은 최악의 경험 있어요?

저는 짐을 공항에 두고 온 적이...😭
댓글로 당신의 TMI 들려주세요!
```

**의견 수렴:**
```
진지한 질문)
프리랜서 vs 직장인
어떤 게 더 나을까요?

둘 다 경험해보신 분들의
솔직한 의견이 궁금해요.
```

#### **4-2. 스토리텔링 구조**

**기승전결:**
```
[기] 오늘 회사 앞 카페에서 일어난 일
[승] 단골 바리스타님이 제 취향 기억하고
[전] 새로운 메뉴 추천해주셨는데
[결] 인생 커피를 찾았어요 ☕✨

당신만의 단골집 이야기 들려주세요!
```

#### **4-3. 시리즈 콘텐츠**

**넘버링 활용:**
```
📚 직장인 필독서 추천 (1/5)

오늘은 '아무튼, 출근'
월요병 치료제 같은 책...

(2편이 기대되는 구조로 팔로우 유도)
```

#### **4-4. 논란/찬반 주제**

```
🔥 논란) 회사에서 점심은 혼자 먹기 vs 같이 먹기

혼자파: 자유롭고 편함
같이파: 관계 형성에 좋음

당신은 어느 쪽?
이유도 함께 댓글로!
```

### Step 5: 콘텐츠 생성 프로세스

#### **5-1. 입력 정보 확인**

사용자가 제공한 정보를 정리하고 부족한 부분은 질문합니다.

**체크리스트:**
- [ ] 주제/키워드 명확한가?
- [ ] 목적이 정의되었는가?
- [ ] 톤 선호도가 있는가?
- [ ] 길이 제한이 있는가?
- [ ] 타겟 오디언스가 명확한가?

#### **5-2. 콘텐츠 초안 생성**

**구조:**
1. **Hook (첫 문장)**: 3초 안에 시선을 끄는 문장
2. **Body (본문)**: 핵심 메시지 전달
3. **CTA (행동 유도)**: 댓글/공유 유도 질문
4. **해시태그**: 3-5개 (본문 또는 댓글)

**예시 템플릿:**
```
[Hook] 💡 [충격적 사실/질문]

[Body]
[짧은 문단 1]
[짧은 문단 2]
[짧은 문단 3]

[CTA] [질문/의견 요청] 😊

#해시태그1 #해시태그2 #해시태그3
```

#### **5-3. 알고리즘 최적화 체크**

생성된 콘텐츠를 다음 기준으로 검증합니다:

**✅ 체크리스트:**
- [ ] 첫 문장이 강력한가?
- [ ] 이모지 3-5개 사용했는가?
- [ ] 문장이 짧고 명확한가? (15-25자)
- [ ] 단락 구분이 명확한가?
- [ ] CTA가 자연스러운가?
- [ ] 해시태그가 3-5개인가?
- [ ] 한국 Threads 톤이 적용되었는가?
- [ ] 참여 유도 요소가 있는가?

#### **5-4. A/B 버전 제공 (선택사항)**

같은 주제로 2-3가지 버전을 생성하여 사용자가 선택하거나 테스트할 수 있게 합니다.

**버전 차이:**
- 버전 A: 짧고 임팩트 있는 스타일
- 버전 B: 스토리텔링 중심
- 버전 C: 질문형 참여 유도

## Technical Details

### Threads 알고리즘 핵심 요소

#### **1. 참여율 (Engagement Rate)**

**가중치 순서:**
1. **댓글** (가장 높음) - 대화 시작을 의미
2. **재게시** (두 번째) - 콘텐츠 확산력
3. **좋아요** (세 번째) - 기본 반응
4. **조회수** (네 번째) - 수동적 지표

**최적화 전략:**
- 댓글을 유도하는 질문 필수
- 재게시 유도: "공감하면 공유해주세요"
- 좋아요 유도: "동의하면 ❤️"

#### **2. 게시 타이밍**

**한국 사용자 기준 골든 타임:**
- **평일 점심시간**: 12:00-13:30
- **평일 퇴근시간**: 18:00-20:00
- **주말 오전**: 10:00-12:00
- **주말 저녁**: 20:00-22:00

**피해야 할 시간:**
- 새벽 1:00-7:00 (낮은 활동)
- 평일 오전 9:00-11:00 (업무 중)

#### **3. 첫 30분 골든타임**

Threads 알고리즘은 **게시 후 30분 내 반응**을 중요하게 평가합니다.

**전략:**
- 골든 타임에 게시
- 게시 직후 직접 댓글 달기 (대화 시작)
- 들어오는 댓글에 빠르게 응답

#### **4. 콘텐츠 길이**

**최적 길이:**
- **단문 (50-150자)**: 빠른 소비, 높은 완독률
- **중문 (150-300자)**: 정보 전달 + 참여 유도
- **장문/스레드 (300자+)**: 심층 콘텐츠, 저장률 높음

**권장:**
- 일반 게시글: 100-200자
- 정보성 게시글: 200-300자
- 스레드: 필요시 여러 개로 분할

#### **5. 비주얼 요소**

Threads는 텍스트 중심이지만 시각적 요소도 중요합니다:

**이모지 활용:**
- 단락 시작에 이모지로 포인트
- 리스트 형태로 이모지 사용

**줄바꿈:**
- 3-4줄마다 줄바꿈
- 가독성 극대화

**예시:**
```
💡 오늘의 팁

✅ 아침 루틴
✅ 물 마시기
✅ 스트레칭

당신의 루틴은?
```

## When to Use

### 이 Skills를 사용해야 할 때

**✅ 사용 권장:**
1. Threads 게시글을 작성할 때
2. 소셜 미디어 콘텐츠를 Threads 형식으로 변환할 때
3. 브랜드 메시지를 친근하게 전달하고 싶을 때
4. 높은 참여율(댓글, 공유)을 목표로 할 때
5. 한국 Threads 사용자에게 최적화된 콘텐츠가 필요할 때
6. 실시간 트렌드에 빠르게 반응해야 할 때
7. 커뮤니티 형성 및 팔로워 소통을 강화하고 싶을 때

### 이 Skills를 사용하지 말아야 할 때

**❌ 사용 비권장:**
1. 긴 형식의 블로그 포스트 (다른 플랫폼 적합)
2. 공식적이고 격식 있는 공지사항 (Threads 톤과 불일치)
3. 완전 자동화된 봇 게시글 (진정성 부족)
4. 해외 영어권 Threads (한국 특화 Skills)
5. 논문이나 기술 문서 (플랫폼 부적합)

### 다른 Skills와의 조합

**추천 조합:**
- **이미지 생성 Skills** + Threads 콘텐츠 = 비주얼 강화
- **트렌드 분석 Skills** + Threads 콘텐츠 = 실시간 대응
- **브랜드 가이드라인 Skills** + Threads 콘텐츠 = 일관성 유지

## Examples

### Example 1: 일상 공감 콘텐츠

**입력:**
```
주제: 월요일 출근
목적: 공감 유도
톤: 친근하고 유머러스
```

**출력:**
```
😭 월요일 아침 알람 소리가
세상에서 제일 무서운 소리인 거 인정?

나: 5분만 더...
현실: 이미 지각 위기 🚨

그래도 오늘 하루도
파이팅 해봅시다 여러분...💪

당신의 월요병 극복법은?

#월요일 #직장인 #공감 #일상
```

### Example 2: 정보 제공 콘텐츠

**입력:**
```
주제: Threads 알고리즘 팁
목적: 정보 전달
톤: 전문적이지만 친근하게
```

**출력:**
```
📌 Threads 노출 폭발시키는 3가지 팁
```

## Best Practices

### ✅ 해야 할 것

1. **Hook 강력하게:** 첫 문장으로 3초 안에 시선을 사로잡기
2. **짧은 문장:** 15-25자 단위로 끊어서 가독성 확보
3. **이모지 적절히:** 3-5개로 시각적 포인트 주기
4. **CTA 필수:** 댓글 유도 질문으로 마무리
5. **해시태그 전략적:** 트렌딩 + 니치 + 브랜드 조합
6. **톤 일관성:** 한국 Threads 문화에 맞는 톤 유지
7. **타이밍:** 골든 타임에 게시
8. **빠른 응답:** 댓글에 즉시 반응하여 알고리즘 부스트

### ❌ 하지 말아야 할 것

1. **너무 긴 글:** 스크롤 피로 유발
2. **이모지 과다:** 스팸처럼 보임
3. **해시태그 도배:** 10개 이상 사용
4. **광고성 멘트:** "구매하세요", "링크 클릭" 등 직접 판매
5. **복잡한 문장:** 전문 용어 남발
6. **CTA 없음:** 일방적 정보 전달만
7. **논란 조장:** 지나친 혐오나 선동
8. **복사 붙여넣기:** 다른 플랫폼 콘텐츠 그대로 사용

## Troubleshooting

### 문제 1: 게시글이 노출되지 않음

**증상:** 조회수가 매우 낮음 (10-20회)

**원인 체크:**
- [ ] 해시태그가 너무 많거나 없는가?
- [ ] 첫 30분 내 참여가 없었는가?
- [ ] 게시 시간이 비활성 시간대인가?
- [ ] 스팸성 단어를 사용했는가?

**해결책:**
- 해시태그 3-5개로 조정
- 골든 타임(점심, 퇴근 시간)에 재게시
- 게시 직후 직접 댓글 달아 대화 시작
- 스팸 필터 단어 제거 ("구매", "클릭", "무료" 등)

### 문제 2: 댓글이 달리지 않음

**증상:** 조회수는 있지만 댓글 0개

**원인 체크:**
- [ ] CTA(질문)가 없는가?
- [ ] 질문이 너무 어렵거나 모호한가?
- [ ] 일방적 정보 전달만 했는가?
- [ ] 공감 요소가 부족한가?

**해결책:**
- 명확하고 쉬운 질문 추가
- 양자택일 질문으로 변경 ("A vs B")
- 개인 경험 공유 요청
- 공감 유도 요소 강화

### 문제 3: 톤이 어색함

**증상:** 딱딱하거나 부자연스러운 느낌

**원인 체크:**
- [ ] 존댓말과 반말이 섞였는가?
- [ ] 번역체 문장인가?
- [ ] 이모지가 너무 없거나 많은가?
- [ ] 한국 Threads 톤이 아닌가?

**해결책:**
- 일관된 말투 선택 (반말 또는 존댓말)
- 자연스러운 구어체로 변경
- 이모지 3-5개로 조정
- KOREAN_THREADS_GUIDE.md 참조
