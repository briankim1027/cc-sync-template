# 한국 Threads 문화 완전 가이드

## 1. 언어 특성

### 1-1. 반말 vs 존댓말

**반말 사용 적합:**
- 개인 크리에이터
- 친근한 브랜드 (카페, 패션)
- 일상 공유 콘텐츠
- 20-30대 타겟

**예시:**
```
"오늘 카페 갔는데 진짜 대박이었어 ㅋㅋㅋ"
```

**존댓말 사용 적합:**
- 기업 공식 계정
- 전문 서비스 (금융, 의료)
- 정보 제공 콘텐츠
- 40대 이상 타겟

**예시:**
```
"오늘 새로운 서비스를 소개해드립니다."
```

### 1-2. 한국 인터넷 용어

**필수 용어:**
- ㅋㅋㅋ (웃음)
- ㅠㅠ (슬픔/안타까움)
- ㄹㅇ (리얼, 진짜)
- 레전드 (전설적인)
- 갓 (최고의)
- TMI (Too Much Information)
- 어쩔 (어쩔 수 없다)
- 인정 (동의)
- 실화냐 (정말이냐)
- 존맛 (매우 맛있다)

**사용 예시:**
```
"이거 ㄹㅇ 레전드 아니냐고 ㅋㅋㅋ"
"TMI) 저는 치즈 버거만 10개 먹어봤어요"
"이 맛집 존맛 인정? ㅇㅈ"
```

## 3. 문화적 특징

### 3-1. 공감 문화

한국 Threads 사용자는 "나도 그래" 공감을 매우 중요하게 생각합니다.

**공감 유도 문구:**
- "저만 그런가요?"
- "다들 이러시나요?"
- "공감하시면 좋아요!"
- "맞죠? 맞죠?"
- "이거 나만 느끼는 거 아니지?"
- "다들 어때요?"

**예시:**
```
"회사에서 점심 혼자 먹는 거
저만 편한가요? ㅋㅋㅋ

다들 어떠세요?"
```

### 3-2. 솔직함과 TMI

너무 완벽하거나 격식 있는 것보다 **솔직하고 친근한** 콘텐츠를 선호합니다.

**예시:**
```
❌ "오늘 프로젝트를 성공적으로 마무리했습니다."

✅ "오늘 프로젝트 끝냈는데 솔직히 기적 같음 ㅋㅋ
   중간에 컴퓨터 꺼져서 3시간치 날아간 건 비밀...
   
   근데 왠지 뿌듯하네요 😊"
```

### 3-3. 자기 비하 유머

한국 특유의 겸손 문화로 인해 자기 비하 유머가 공감을 얻습니다.

**예시:**
```
"저는 운동 시작한 지 3일 만에 포기했어요...
(사실 운동복만 샀음)

근데 운동복은 예쁘니까 됐다! 🏃‍♀️"
```

**주의사항:**
- 너무 심한 자기 비하는 오히려 역효과
- 유머러스하게, 가볍게 풀어내기

### 3-4. 빨리빨리 문화

한국인은 빠른 정보 습득을 선호합니다.

**전략:**
- 핵심을 앞에 배치
- 간결한 문장
- 결론 먼저 제시

**예시:**
```
✅ "결론: 이 카페 존맛

이유는요...
1. 커피 진하고
2. 디저트 맛있고
3. 인테리어 예쁨

위치는 댓글에!"
```

## 6. 이모지 활용

### 6-1. 한국에서 인기 있는 이모지

**감정 표현:**
- 😊 (미소) - 친근함
- 💙 (파란 하트) - 감사, 공감
- ✨ (반짝임) - 특별함
- 💕 (두 개의 하트) - 사랑스러움
- 🥺 (애교) - 귀여움, 부탁

**강조:**
- ⭐ (별) - 중요
- 💡 (전구) - 팁, 아이디어
- 🔥 (불) - 핫한, 인기
- ✅ (체크) - 완료, 동의
- 📌 (핀) - 고정, 중요

**일상:**
- ☕ (커피) - 카페, 여유
- 🍔 (햄버거) - 맛집
- 📱 (스마트폰) - 디지털
- 💻 (노트북) - 업무
- 🎵 (음악) - 취미

**자연:**
- 🌸 (벚꽃) - 봄
- 🌙 (달) - 밤, 감성
- ☀️ (태양) - 아침, 활기
- 🌿 (나뭇잎) - 자연, 힐링

### 6-2. 이모지 조합 패턴

**정보 제공 시:**
```
📌 핵심 포인트
💡 꿀팁
✅ 체크리스트
```

**일상 공유 시:**
```
☕ 카페 방문
🍔 맛집 탐방
📱 일상 기록
```

**감성 표현 시:**
```
✨ 특별한 순간
💙 감사한 마음
🌸 봄날의 감성
```

## 10. 성공을 위한 핵심 원칙

### 10가지 원칙

1. **진정성**: 꾸미지 않은 솔직함
2. **공감**: "나도 그래" 유도
3. **간결함**: 핵심만 간단히
4. **시각성**: 이모지 활용
5. **타이밍**: 골든 타임 게시

8. **트렌드**: 최신 이슈 반영
9. **가치**: 유익한 정보
10. **재미**: 즐거운 콘텐츠
