# 짧은 게시글 템플릿

## 목적
빠르게 소비 가능한 50-100자 내외의 임팩트 있는 게시글

## 구조

```
[Hook 이모지] [강력한 첫 문장]

[핵심 메시지 1-2줄]

[CTA 질문] [이모지]

#해시태그1 #해시태그2 #해시태그3
```

## 예시 1: 공감형

```
😭 월요일 아침 알람 소리가
세상에서 제일 무서운 소리 인정?

당신의 월요병 극복법은? 💬

#월요일 #직장인 #공감
```

## 예시 2: 팁 제공형

```
💡 커피 주문 꿀팁!

"얼음 적게" 대신
"얼음 빼고 컵에 꽉 채워주세요"

이거 아는 사람? ✋

#커피 #꿀팁 #카페
```

## 예시 3: 질문형

```
☕ vs 🍵

아침에 커피 vs 차
당신의 선택은?

댓글로 투표! 👇

#아침루틴 #커피 #차
```

## 작성 가이드

### ✅ 해야 할 것
- 첫 문장에 이모지로 시선 집중
- 핵심만 간결하게
- 명확한 CTA
- 3-5개 해시태그

### ❌ 하지 말아야 할 것
- 불필요한 설명 추가
- 복잡한 문장
- CTA 없이 끝내기
- 해시태그 과다

## 최적 사용 시기
- 빠른 반응이 필요할 때
- 간단한 질문/투표
- 일상 공유
- 트렌드 빠른 반응
