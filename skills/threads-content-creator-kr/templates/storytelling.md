## 예시 1: 일상 감성

```
오늘 카페에서 겪은 일 💙

단골 카페 바리스타님이
제 취향 기억하고 계시더라구요.

"설탕 안 넣으시죠?"
"따뜻한 거 좋아하시죠?"

별거 아닌 것 같지만
나를 기억해준다는 게
이렇게 따뜻할 수가 없어요. ☕✨

작지만 소중한 순간들이
쌓여서 인생이 되는 것 같아요.

당신의 따뜻했던 순간 공유해주세요!

#일상 #소확행 #감성 #카페
```

## 예시 2: 교훈 담긴 이야기

```
3년 전 퇴사를 결심했을 때 💭

주변 모두가 말렸어요.
"안정적인 직장인데"
"나중에 후회할 거야"

솔직히 무서웠죠.
하지만 더 무서운 건
후회하며 사는 거였어요.

지금은요?
최고의 결정이었다고
자신 있게 말할 수 있어요. 🌟

물론 쉽지 않았지만
제 선택을 믿었던 게
가장 잘한 일이에요.

당신이 용기 낸 순간이 있나요?

#퇴사 #용기 #인생 #선택
```

## 예시 3: 공감 에피소드

```
어제 지하철에서 본 광경 🚇

할머니가 젊은이에게 자리 양보 받고
"고마워요, 착한 사람"

그 젊은이는 다리 깁스하고 있었어요.
할머니 그걸 보더니
다시 자리 권했고

둘이 서로 앉으라고
티격태격하는 모습이
너무 웃기면서도 따뜻했어요. 😊

세상은 아직 살만한 것 같아요.

당신이 본 따뜻한 광경 들려주세요!

#일상 #훈훈 #지하철 #감동
```
