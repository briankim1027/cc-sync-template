# 시리즈 콘텐츠 템플릿

## 목적
연속성 있는 콘텐츠로 팔로워 유지 및 기대감 형성

## 구조

```
[시리즈 이모지] [시리즈 제목] (N/총개수)

[오늘의 주제]

[핵심 내용]
- 포인트 1
- 포인트 2
- 포인트 3

[다음 편 예고]

[팔로우 유도]

#해시태그1 #해시태그2 #시리즈명
```

## 예시 1: 정보 시리즈

```
📱 직장인 생산성 앱 추천 (1/5)

오늘은 "노션 Notion"

왜 필수냐면,
✅ 업무 정리
✅ 프로젝트 관리
✅ 개인 위키
✅ 협업 도구

저는 회의록, 아이디어, 일정
전부 노션에 몰아넣어요.

처음엔 복잡해 보이지만
2주만 쓰면 중독됨 주의! 😂

📌 내일은 시간 관리 앱 소개!
- 타이머 기능
- 집중 모드
- 통계 분석

기대되면 팔로우 😊
당신의 필수 앱은 뭔가요?

#직장인앱 #생산성 #노션 #업무팁
```

## 예시 3: 스토리 시리즈

```
면접 팁 궁금한 거 댓글로 물어보세요!

#이직 #면접 #취준생 #커리어
```

## 작성 가이드

### ✅ 해야 할 것
- 명확한 넘버링 (N/총개수)
- 일관된 시리즈명
- 다음 편 예고
- 이전 편 간단 요약
- 정기성 유지

### ❌ 하지 말아야 할 것
- 불규칙한 업로드
- 시리즈명 변경
- 예고 없이 중단
- 너무 긴 시리즈 (10개 이상)

## 시리즈 타입

### 1. 정보 시리즈
- "TOP N 시리즈"
- "완전 정복 시리즈"
- "A to Z 가이드"

### 2. 챌린지 시리즈
- "N일 챌린지"
- "함께 하는 OO"
- "데일리 미션"

### 3. 스토리 시리즈
- "나의 OO 다이어리"
- "실전 경험담"
- "비하인드 스토리"

### 4. 정기 시리즈
- "요일별 추천"
- "주간 정리"
- "매일매일 팁"

## 성공 포인트

1. **일관성**: 같은 시간, 같은 요일
2. **예고**: 다음 편 기대감 형성
3. **완결**: 끝까지 완주
4. **참여**: 댓글로 소통
5. **보상**: 완주자 혜택

## 최적 사용 시기
- 장기 콘텐츠 계획
- 팔로워 유지 목적
- 전문성 구축
- 커뮤니티 형성
- 브랜드 구축
