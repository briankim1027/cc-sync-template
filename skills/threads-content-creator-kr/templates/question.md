# 질문형 템플릿

## 목적
댓글과 참여를 극대화하는 질문 중심 게시글

## 구조

```
[주제 이모지] [흥미로운 주제 제시]

[선택지 A]
[설명/이유]

[선택지 B]
[설명/이유]

[개인 의견 (선택)]

[참여 유도 문구]

#해시태그1 #해시태그2 #해시태그3
```

## 예시 1: 양자택일

```
🏠 vs 🏢 논란 주의!

재택근무 vs 사무실 출근
당신의 선택은?

재택파 👇
"집에서 편하게 일하는 게 최고"
"출퇴근 시간 아끼고 생산성 UP"

사무실파 👇
"동료들이랑 소통하면서 일해야 집중"
"집에선 침대 유혹이..."

저는 하이브리드가 제일 좋더라구요!

당신의 이상적인 근무 형태는?
이유와 함께 댓글로! 💬

#재택근무 #워라밸 #직장인
```


## 작성 가이드

### ✅ 해야 할 것
- 명확한 선택지 제시
- 각 선택지 설명
- 답하기 쉬운 구조
- 중립적 톤 유지
- 개인 의견 먼저 공유

### ❌ 하지 말아야 할 것
- 너무 복잡한 질문
- 편향된 선택지
- 민감한 주제 (정치/종교)
- 개인정보 요구
- 판단 강요

## 참여율 높이는 팁

1. **쉬운 질문**: A/B 선택 또는 단답형
2. **공감대**: 누구나 경험한 주제
3. **개인화**: "당신은?" 직접 호명
4. **보상**: 통계 공유, 결과 발표
5. **시간 제한**: "오늘까지만!" 긴박감

## 최적 사용 시기
- 점심시간 (12-14시)
- 퇴근시간 (18-20시)
- 활발한 의견 교류 원할 때
- 커뮤니티 형성 목적
