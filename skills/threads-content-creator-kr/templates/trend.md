# 트렌드 반응 템플릿

## 목적
실시간 이슈나 트렌드에 빠르게 반응하여 높은 노출 달성

## 구조

```
[트렌드 이모지] [화제의 키워드]

[트렌드 요약]

[개인 의견/해석]

[연결점 찾기]
- 나의 경험
- 일상과 연결
- 독자와 관계

[의견 묻기]

#트렌딩해시태그 #관련태그
```

```

3. OO 대비책

댓글로 의견 나눠요!

#OO사건 #실시간 #이슈
```

## 작성 가이드

### ✅ 해야 할 것
- 빠른 반응 (이슈 발생 후 2-6시간)
- 간단한 요약 제공
- 개인 의견 명확히
- 중립적 관점 유지 (논란 주제)
- 트렌딩 해시태그 활용

### ❌ 하지 말아야 할 것
- 확인 안 된 정보 공유
- 편향적 의견 강요
- 민감한 주제 자극
- 혐오 표현
- 상업적 이용만

## 트렌드 찾는 방법

### 1. 실시간 검색어
- 포털 사이트
- SNS 트렌드
- 뉴스 헤드라인

### 2. Threads 내부
- 인기 해시태그
- 많이 언급되는 주제
- 팔로잉 계정 반응

### 3. 주변 관찰
- 사람들 대화 주제
- 카페/지하철 풍경
- 매장 진열 상품

## 트렌드 타입별 전략

### 속보성 이슈 (당일)
- 즉각 반응
- 간단한 의견
- 빠른 업로드

### 화제성 이슈 (1-3일)
- 심층 분석
- 다양한 관점
- 연결점 찾기

### 시즌 트렌드 (주/월)
- 미리 준비
- 시리즈화
- 지속 업로드

## 안전 가이드

### 피해야 할 이슈
- 정치 논쟁
- 종교 갈등
- 재난/사고 (조심)
- 명예훼손 소지
- 혐오 조장

### 안전한 접근
- 사실 확인 필수
- 출처 명시
- 중립적 표현
- 다양한 관점 제시
- 존중하는 태도

## 타이밍 전략

### 골든 타임
- 이슈 발생 후 2-6시간
- 검색량 상승기
- 아직 포화 전

### 피크 타임
- 점심시간 (12-14시)
- 퇴근시간 (18-20시)
- 주말 저녁 (20-22시)

### 피해야 할 시간
- 심야 (23시-6시)
- 업무 시간 (10-12시)
- 새벽 (1-5시)

## 성공 포인트

1. **속도**: 빠른 반응
2. **간결**: 핵심만
3. **관점**: 독특한 시각
4. **연결**: 일상과 연결
5. **해시태그**: 트렌딩 활용

## 예상 효과

### 즉각 효과
- 높은 조회수
- 빠른 참여
- 새 팔로워 유입

### 장기 효과
- 트렌드 민감 계정 이미지
- 활발한 계정 인식
- 알고리즘 선호

## 주의사항

1. **팩트 체크**: 확인 후 게시
2. **조심스런 주제**: 신중하게
3. **출처 명시**: 정보 출처
4. **존중**: 다양한 의견
5. **책임감**: 영향력 인식

트렌드에 현명하게 반응하여
Threads에서 영향력을 키워보세요!
