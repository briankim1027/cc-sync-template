## 3. 참여 유도 기법

### 3-1. 갭 기법

정보의 일부만 제공하고 나머지는 댓글이나 다음 게시글에서 공개.

**예시 1: 넘버링 티저**
```
"직장 생활 10년 차가 알려주는
절대 하면 안 되는 실수 3가지

1. [비공개]
2. [비공개]
3. [비공개]

댓글로 궁금한 번호 적어주세요!
가장 많이 나온 거부터 공개할게요 😊"
```

**예시 2: 결말 보류**
```
"오늘 카페에서 진짜 황당한 일이...

(댓글 30개 넘으면 공개합니다 ㅋㅋ)"
```

**예시 3: 정보 분할**
```
"연봉 협상 꿀팁 TOP 5

오늘은 1-2위까지만!
3-5위는 댓글 반응 보고 내일 올릴게요 💰"
```

### 3-2. 투표/퀴즈 기법

**예시 1: OX 퀴즈**
```
"🎯 OX 퀴즈!

'아메리카노'는 이탈리아어로
'미국식 커피'라는 뜻이다.

O 라고 생각하면 ⭕ 댓글
X 라고 생각하면 ❌ 댓글

정답은 1시간 후 공개!"
```

**예시 2: 순위 맞추기**
```
"한국인이 가장 많이 마시는 커피 TOP 3
순서를 맞춰보세요!

☕ 카페모카

댓글로 순위 적어주세요!

```

```
B) 5,500원
C) 6,500원

댓글로 답변! 정답자 중 추첨 🎁"
```

## 5. 심리 기법 활용

### 5-1. FOMO (Fear of Missing Out)

놓치면 안 되는 정보라는 심리 활용

**예시:**
```
"⚠️ 이거 모르면 진짜 손해

내일까지만 공유할게요
놓치지 마세요!"
```

### 5-2. 사회적 증거

많은 사람이 하고 있다는 증거 제시

**예시:**
```
"벌써 1,000명이 저장한 꿀팁!

당신도 저장해두세요 📌"
```

### 5-3. 호혜성의 법칙

먼저 가치를 제공하고 참여 유도

**예시:**
```
"무료로 템플릿 공유합니다!

도움 됐으면 댓글로 피드백 부탁드려요 💙"
```

### 5-4. 희소성

한정된 기회 강조

**예시:**
```
"선착순 10명만!

댓글 남기신 분께 [혜택] 드려요"
```
