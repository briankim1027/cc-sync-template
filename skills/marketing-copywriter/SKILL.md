# Marketing Copywriter

## Overview

Transform product or service descriptions into high-converting marketing copy by synthesizing the timeless principles of history's greatest copywriting masters. This skill integrates frameworks from David Ogilvy (research-driven brand building), Claude Hopkins (scientific specificity), Gary Halbert (emotional connection), Eugene Schwartz (awareness states), Joe Sugarman (psychological triggers), and John Caples (tested formulas).

Generate persuasive copy for landing pages, social media advertisements, email campaigns, and sales materials that speak directly to your audience's desires and drive measurable action.

## When to Use This Skill

Invoke this skill when:

- Creating landing page copy (headlines, subheadlines, CTAs, benefit sections)
- Writing social media advertisements (Facebook, Instagram, LinkedIn ads)
- Developing email marketing campaigns
- Crafting product descriptions that convert
- Building sales pages or sales letters
- Testing different copy variations for A/B testing
- Need to transform features into compelling benefits
- Require proven psychological triggers for persuasion
- Want copy analyzed against master copywriter principles

**Trigger phrases:** "write landing page copy," "create ad copy," "generate marketing copy," "write sales copy," "persuasive copy for [product]"

## Core Capabilities

### 1. Multi-Master Copy Generation

Integrate principles from all 6 copywriting legends to create comprehensive, conversion-optimized copy:

**Process:**
1. **Research Phase (Ogilvy):** Understand the product/service thoroughly before writing
2. **Awareness Assessment (Schwartz):** Identify target audience's awareness level
3. **Specificity Focus (Hopkins):** Use concrete facts, numbers, and unique mechanisms
4. **Emotional Hooks (Halbert + Sugarman):** Deploy psychological triggers and storytelling
5. **Proven Formulas (Caples):** Apply tested headline and structure templates
6. **The Big Idea (Ogilvy):** Distill everything into one compelling central concept

**Example Usage:**
```
User: "Write landing page copy for a productivity app that uses AI to automatically prioritize tasks"

Claude: [Analyzes product] → [Identifies awareness level] → [Applies frameworks]:

"Stop Drowning in Your To-Do List: How This AI Does Your Prioritizing in 3 Seconds"

SUBHEADLINE (Ogilvy + Schwartz):
"The same intelligent system used by 47,000 professionals to reclaim 15 hours per week—
without touching a single task manually"

[Full landing page structure with integrated principles...]
```

### 2. Copy Type Specialization

**Landing Pages:**
- Hero section (headline, subheadline, CTA)
- Problem-solution structure
- Benefit-driven body sections
- Social proof integration
- FAQ objection handling
- Multi-stage CTA placement

**Social Media Ads:**
- Pattern-interrupt hooks (3-second attention grab)
- Curiosity-driven short copy (150-200 words)
- Benefit bullets with emoji formatting
- Clear, action-oriented CTAs
- Platform-specific optimization (FB/IG/LinkedIn)

**Email Campaigns:**
- Subject line testing variations
- Opening hook optimization

- P.S. urgency tactics

### 3. Psychological Trigger Application

Automatically identify and deploy the most effective triggers for each context:

- **Scarcity & Urgency:** Time/quantity limits
- **Social Proof:** Testimonials, user counts, authority
- **Reciprocity:** Free value offers
- **Fear of Loss:** Cost of inaction framing
- **Curiosity Gaps:** Information loops
- **Authority:** Expert endorsements
- **Simplicity:** Easy implementation promises

Reference: `psychological_triggers.md` for complete trigger library and usage guidelines.

### 4. Framework-Based Structuring

Apply proven copywriting structures:

- **AIDA:** Attention → Interest → Desire → Action
- **PAS:** Problem → Agitate → Solve
- **BAB:** Before → After → Bridge
- **4 Ps:** Picture → Promise → Prove → Push
- **FAB:** Features → Advantages → Benefits

Reference: `frameworks.md` for complete templates and examples.

### 5. Awareness State Adaptation (Eugene Schwartz)

Tailor copy sophistication to audience awareness level:

- **Unaware:** Problem identification focus
- **Problem Aware:** Solution education
- **Solution Aware:** Unique mechanism highlighting
- **Product Aware:** Differentiation and objection handling
- **Most Aware:** Simple offer with strong CTA

### 6. Headline & Hook Generation

Create multiple headline variations using Caples' proven formulas:

- How-to headlines: "How to [Outcome] in [Timeframe]"
- Curiosity headlines: "The Secret to [Benefit]"
- Question headlines: "Do You Make These Mistakes in [Area]?"
- Benefit headlines: "[Achieve Outcome] Without [Pain]"
- News headlines: "Announcing: [New Solution]"

Reference: `frameworks.md` contains 50+ headline formulas.

### 7. Expert-Specific Analysis

Request copy from the perspective of individual masters:

- **"Ogilvy approach"** → Long copy, brand building, research-backed
- **"Hopkins style"** → Reason-why, scientific testing, specific claims
- **"Halbert voice"** → Starving crowd, strong offer, fascination bullets
- **"Schwartz framework"** → Mass desire channeling, unique mechanism
- **"Sugarman flow"** → Slippery slide, psychological triggers
- **"Caples formula"** → Tested headlines, self-interest appeals

## Copy Quality Standards

All generated copy adheres to these master-approved principles:

**✓ Specificity:** Concrete numbers, facts, details (Hopkins)
**✓ Benefits Focus:** What's in it for the reader (Caples)
**✓ Emotional Connection:** Stories and psychological triggers (Sugarman + Halbert)
**✓ Proof Elements:** Social proof, testimonials, guarantees (All masters)
**✓ Clear CTA:** Single, obvious next step (Hopkins + Halbert)
**✓ Authentic Voice:** Conversational, not corporate (Ogilvy + Halbert)
**✓ Unique Mechanism:** The "reason why" it's different (Schwartz)

## Best Practices

**DO:**
- Provide detailed product/service information for better copy
- Specify target audience and their main pain points
- Indicate desired copy format (landing page, ad, email)
- Request multiple variations for testing
- Ask for specific master's approach when needed

**DON'T:**
- Expect generic copy without product details
- Use copy verbatim without adapting to brand voice
- Ignore A/B testing—always test variations
- Forget to include proof elements (testimonials, guarantees)
- Overlook mobile optimization for social ads
