# Consulting & Coaching Service Template

## Post Structure

```
[개인 스토리 Hook - 과거 도전 과제나 전환점]

[공백]

[구체적인 문제 상황 묘사]
- 어떤 어려움이 있었는지
- 왜 중요했는지
- 어떤 영향을 미쳤는지

[공백]

[Journey/Learning 과정]
시도한 것들:
✓ 방법 1
✓ 방법 2
✓ 깨달음

[공백]

[어떻게 다른 사람을 돕고 있는지]
현재 제공하는 서비스:
→ 서비스 1 (구체적 결과)
→ 서비스 2 (독특한 방법론)
→ 서비스 3 (맞춤형 접근)

[공백]

[클라이언트 성공 사례]
최근 클라이언트 결과:
• 성과 1 (구체적 숫자)
• 성과 2 (변화의 폭)
• 평균 만족도 또는 재계약율

[공백]

[Soft CTA - 대화 시작 유도]
비슷한 고민 있으신가요?
DM으로 [키워드] 보내주시면 [제공 내용]

[공백]

#Coaching #Consulting #Leadership #PersonalBranding
```

## Example

```
3년 전, 저는 최고의 마케터였지만 최악의 리더였습니다.

팀원들은 번아웃.
이직률 40%.
프로젝트는 항상 지연.

"뛰어난 개인 기여자"와 "훌륭한 리더"는
완전히 다른 스킬셋이라는 걸 그때 깨달았습니다.

그래서 시작한 것:
✓ 100권의 리더십 책 독파
✓ 20명의 멘토와 1:1 세션
✓ 매일 팀원 피드백 수집

12개월 후:
→ 팀 이직률 5%로 감소
→ 프로젝트 납기 준수율 95%
→ 팀 만족도 4.8/5.0

이제 이 경험을 다른 마케팅 리더들과 공유합니다.

Leadership Coaching Program:
→ 1:1 맞춤 코칭 (주 1회, 12주)
→ 실전 적용 가능한 플레이북
→ 팀 다이나믹스 개선 전략

지난 분기 15명의 리더가:
• 평균 팀 생산성 35% 증가
• 이직률 50% 감소
• 승진 또는 급여 인상 달성

마케팅 리더로서 성장하고 싶으신가요?
DM으로 '성장'이라고 보내주시면
무료 30분 진단 세션을 제공합니다.

#MarketingLeadership #TeamManagement #Coaching #CareerGrowth
```

## Customization Tips

### Hook 옵션:
1. "[시간] 전, 저는 [상황]이었습니다"
2. "저는 [성과]를 달성했지만 [문제]로 고통받았습니다"
3. "[전환점] 순간, 모든 것이 바뀌었습니다"

### Journey 요소:
- 실패와 시행착오 솔직하게 공유
- 구체적인 학습 과정 나열
- 전환점이 된 인사이트 강조

### Success Metrics:
- Before/After 비교 (%, 절대값)
- 클라이언트 수 및 재계약율
- 구체적 성과 (승진, 매출 증가 등)

### CTA 옵션:
- "무료 30분 진단 세션"
- "케이스 스터디 다운로드"
- "월간 뉴스레터 구독"
- "DM으로 [키워드] 보내주세요"
