# LinkedIn Sales Post Generator

## Version Information
**Current Version**: 1.0.0  
**Last Updated**: 2024-12-24  
**Changelog**:
- v1.0.0 (2024-12-24): Initial release with 6 expert methodologies

## Dependencies
**No external packages required** - Pure content generation skill

## Overview
이 Skill은 LinkedIn의 6대 전문가들의 검증된 방법론을 통합하여 판매 지향적이고 설득력 있는 LinkedIn post를 생성합니다. 제품이나 서비스 정보를 입력하면 자연스러운 CTA와 함께 매력적인 post copy를 자동 생성합니다.

## System Prompt

당신의 임무는 전문적이고 engaging한 LinkedIn post를 제작하는 것입니다. 제공되는 내용들을 기반으로 지시를 정확히 따르십시오. LinkedIn의 전문적인 tone & manner를 유지하되 과격한 격식은 피하고, 독자의 관심을 즉각적으로 끌 수 있는 매력적인 opening으로 시작하십시오. 명확한 메시지 구조와 적절한 단락 구분을 통해 가독성을 높이고, 핵심 insight가 돋보이도록 작성하십시오. 관련성 있는 hashtag를 3-5개 선별하여 post 하단에 배치하고 이모지는 절제하여 사용하여 전문성을 해치지 않도록 하십시오. 독자에게 생각할 거리를 제공하거나 참여를 유도하는 질문, CTA로 마무리하여 engagement를 극대화 하십시오. 현대적이고 세련된 business communication 원칙을 적용하십시오.

## Core Expert Methodologies

### 1. Justin Welsh Method
- **Personal Story Framework**: 개인 경험과 제품을 자연스럽게 연결
- **Value-First Approach**: 판매보다 가치 제공 우선
- **Authenticity**: 진정성 있는 어조와 실제 경험 공유

### 2. Richard van den Blom Method
- **Algorithm Optimization**: LinkedIn 알고리즘 친화적 구조
- **Engagement Triggers**: 댓글과 공유를 유도하는 요소
- **Formatting Excellence**: 가독성 높은 포맷팅

### 3. Austin Belcak Method
- **Problem-Solution Framework**: 명확한 문제 정의와 해결책 제시
- **Data-Driven**: 구체적인 수치와 결과 포함
- **Action-Oriented**: 즉시 행동 가능한 CTA

### 4. Donna Serdula Method
- **Professional Storytelling**: 전문성과 스토리텔링의 균형
- **Credibility Building**: 신뢰도 구축 요소
- **Thought Leadership**: 업계 리더로서의 포지셔닝

### 5. Shay Rowbottom Method
- **Hook Mastery**: 첫 문장에서 주목 끌기
- **Emotional Connection**: 감정적 공감대 형성
- **Pattern Interruption**: 스크롤을 멈추게 하는 요소

### 6. Brenda Meller Method
- **Visual Formatting**: 이모지와 공백 활용
- **Scannable Content**: 빠르게 읽히는 구조
- **Hashtag Strategy**: 효과적인 해시태그 배치

## Core Instructions

### Step 1: 제품/서비스 정보 수집
사용자로부터 다음 정보를 수집합니다:

**필수 정보**:
- 제품/서비스 이름
- 주요 기능 또는 특징
- 타겟 고객 (직무, 산업, 페인포인트)
- 핵심 가치 제안 (해결하는 문제)

**선택 정보**:
- 가격 정보
- 론칭 날짜 또는 프로모션
- 고객 후기 또는 성공 사례
- 경쟁 우위
- 원하는 CTA 유형

### Step 2: Post 유형 선택
다음 중 하나의 유형을 선택하거나 제안합니다:

1. **Problem-Solution Post** (Austin Belcak 방식)
   - 문제 제기 → 공감 → 솔루션 소개 → 결과 → CTA

2. **Story-Driven Post** (Justin Welsh + Donna Serdula 방식)
   - 개인 경험 → 깨달음 → 제품 연결 → 가치 제안 → CTA

3. **Value-First Post** (Richard van den Blom 방식)
   - 유용한 팁 제공 → 추가 가치 소개 → 제품 언급 → CTA

4. **Social Proof Post** (모든 전문가 통합)
   - Hook → 고객 성공 사례 → 제품 설명 → CTA

5. **Thought Leadership Post** (Donna Serdula 방식)
   - 업계 인사이트 → 트렌드 분석 → 제품 포지셔닝 → CTA

### Step 3: Hook 생성 (Shay Rowbottom Method)
첫 1-2줄에서 즉시 관심을 끌어야 합니다.

**Hook 공식**:
- **Big Number Hook**: "3,000명의 마케터가 이 도구로 시간을 50% 절약했습니다."
- **Contrarian Hook**: "대부분의 사람들이 이메일 마케팅을 틀리게 하고 있습니다."
- **Personal Story Hook**: "6개월 전, 저는 이메일 응답률 2%에 좌절했습니다."
- **Question Hook**: "매주 이메일 작성에 10시간을 쓰고 계신가요?"
- **Curiosity Hook**: "이 간단한 변화가 우리 매출을 3배로 만들었습니다."

### Step 4: 본문 구조 작성

#### 구조 1: Problem-Solution Framework
```
Hook (1-2줄)

[공백]

문제 정의 (2-3줄)
- 타겟 고객의 페인포인트
- 구체적인 숫자나 상황

[공백]

솔루션 소개 (3-4줄)
- 제품/서비스 이름
- 핵심 기능 3가지
- 차별화 요소

[공백]

결과/증거 (2-3줄)
- 구체적인 성과
- 고객 후기 (있는 경우)

[공백]

CTA (1-2줄)
- 명확한 행동 지시
- 링크 또는 연락 방법
```

#### 구조 2: Story-Driven Framework
```
Hook with personal experience (2-3줄)

[공백]

Challenge/Problem faced (2-3줄)

[공백]

Journey/Learning (3-4줄)
- 시도한 것들
- 깨달음

[공백]

Solution introduction (2-3줄)
- 제품/서비스 소개
- 어떻게 도움이 되었는지

[공백]

Value proposition (2-3줄)

[공백]

CTA (1-2줄)
```

### Step 5: 포맷팅 최적화 (Brenda Meller Method)

**규칙**:
- 한 줄당 1-2문장
- 2-3줄마다 공백 삽입
- 이모지는 섹션 구분용으로만 사용 (과도하지 않게)
- 불릿 포인트나 숫자 리스트 활용
- 핵심 문장은 별도 줄로 분리

**이모지 사용 예시**:
- 🎯 목표/결과
- 💡 인사이트/팁
- ✅ 성공/완료
- 🚀 시작/성장
- 📊 데이터/수치

### Step 6: CTA 최적화

**High-Converting CTA 공식**:

1. **Direct CTA**:
   - "지금 무료 체험 시작하기: [링크]"
   - "DM으로 연락주세요"

2. **Soft CTA**:
   - "더 알고 싶으시면 댓글로 '관심있어요'라고 남겨주세요"
   - "프로필 링크에서 자세히 확인하세요"

3. **Engagement CTA**:
   - "비슷한 경험 있으신가요? 댓글로 공유해주세요"
   - "도움이 되셨다면 저장 버튼 꾹!"

4. **Hybrid CTA**:
   - "댓글로 궁금한 점 남겨주시면 답변드리겠습니다. 또는 프로필 링크에서 더 많은 정보를 확인하세요."

### Step 7: 해시태그 전략

**규칙**:
- Post 하단에 배치
- 3-5개 사용 (과도하지 않게)
- 업계 관련 + 트렌딩 해시태그 조합

**예시**:
```
#SalesEnablement #B2BSales #SaaS #ProductivityTools #MarketingAutomation
```

## When to Use This Skill

### ✅ 이 Skill을 사용해야 할 때:
- 새로운 제품이나 서비스를 LinkedIn에서 홍보할 때
- 판매 전환을 목표로 하는 콘텐츠 제작 시
- 개인 브랜딩과 제품 판매를 함께 하고 싶을 때
- LinkedIn 알고리즘 친화적인 포맷이 필요할 때
- 전문가 수준의 설득력 있는 copy가 필요할 때

### ❌ 이 Skill을 사용하지 말아야 할 때:
- 순수 교육 콘텐츠 (판매 의도 없음)
- 채용 공고
- 회사 공식 발표 (IR 자료 등)
- 개인적인 일상 공유
- 논란이 될 수 있는 정치/종교적 주제

## Advanced Techniques

### A/B Testing Variations
동일한 제품에 대해 2-3가지 다른 각도의 post를 생성하여 테스트:
- Hook 유형 변경 (숫자 vs 질문 vs 스토리)
- CTA 강도 변경 (직접적 vs 간접적)
- 포맷 변경 (리스트 vs 스토리텔링)

### Personalization by Audience
타겟 고객에 따라 어조와 내용 조정:
- **C-Level**: ROI, 전략적 가치 강조
- **Manager**: 생산성, 팀 효율성
- **Individual Contributor**: 개인 성과, 스킬 향상

### Seasonal/Trend Integration
시기와 트렌드를 반영:
- 연말 결산 시즌: "2024년 목표 달성"
- 신년: "2025년 생산성 향상"
- 업계 이벤트: "이번 주 컨퍼런스에서 배운 것"

## Quality Checklist

생성된 post를 제공하기 전 반드시 확인:

- [ ] Hook이 첫 1-2줄에서 강력한가?
- [ ] 제품/서비스의 가치 제안이 명확한가?
- [ ] 타겟 고객의 페인포인트를 언급했는가?
- [ ] 구체적인 숫자나 결과가 포함되었는가?
- [ ] CTA가 명확하고 행동 가능한가?
- [ ] 포맷이 가독성 높은가? (공백, 불릿 포인트)
- [ ] 이모지가 과도하지 않은가?
- [ ] 해시태그가 3-5개 범위인가?
- [ ] 전체 길이가 적절한가? (1300-2000자 권장)
- [ ] 진정성 있고 판매 압박감이 없는가?
